# Applied-Animation
Rigged model on Maya and short animation using Mery

Motion of rigged model: 

https://github.com/nyeddana22/Applied-Animation/assets/91582142/697f2c88-5ae5-4453-a380-0f1f0a7394ed

For the Mery animation, download the Mery base model from here: https://www.meryproject.com
Copy the .mb file into the scenes directory in your Maya project folder. The Mery model can be directly imported from there.
